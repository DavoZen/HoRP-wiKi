# Тестування GitHub Flavored Markdown

Ця сторінка демонструє всі функції GitHub Flavored Markdown (GFM), які підтримуються у HoRP-wiKi.

## 1. Форматування тексту

### Жирний та курсив
- **Жирний текст** або __жирний текст__
- *Курсивний текст* або _курсивний текст_
- ***Жирний курсив*** або ___жирний курсив___
- ~~Закреслений текст~~

### Верхній та нижній індекси
- Верхній індекс: E = mc<sup>2</sup>
- Нижній індекс: H<sub>2</sub>O

## 2. Заголовки

# Заголовок 1 рівня
## Заголовок 2 рівня
### Заголовок 3 рівня
#### Заголовок 4 рівня
##### Заголовок 5 рівня
###### Заголовок 6 рівня

## 3. Списки

### Нумеровані списки
1. Перший пункт
2. Другий пункт
3. Третій пункт

### Марковані списки
- Пункт 1
- Пункт 2
- Пункт 3

### Вкладені списки
1. Основний пункт
   - Вкладений пункт
   - Ще один вкладений
2. Наступний основний
   1. Вкладений нумерований
   2. Ще один

### Списки завдань (чеклисти)
- [ ] Незавершене завдання
- [x] Завершене завдання
- [ ] Ще одне незавершене

## 4. Блокові цитати

> Це звичайна цитата
>
> Друга лінія цитати

> Цитата першого рівня
>> Вкладена цитата другого рівня
>>> Цитата третього рівня

## 5. Горизонтальні лінії

---

Три тире

***

Три зірочки

___

Три підкреслення

## 6. Код

### Інлайн код
Використовуйте функцію `console.log()` для виведення в консоль.

### Блоки коду
```javascript
function helloWorld() {
    console.log("Привіт, світ!");
    return "Hello, World!";

```

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```
